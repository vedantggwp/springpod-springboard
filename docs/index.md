# Vibe Coding Framework (VCF)

**A system for building AI-powered tools — quickly, safely, and consistently.**

VCF guides you through every step of building with AI coding tools so you don't need to be a developer or memorize every rule. The framework handles that for you.

---

## How it works

<div class="grid cards" markdown>

- :material-clipboard-text:{ .lg .middle } **1. Tell us what you're building**

    ---

    Answer 5 quick questions about your project. The framework picks the right level of guidance.

    [:octicons-arrow-right-24: Project Intake](intake/project-intake.md)

- :material-map-marker-path:{ .lg .middle } **2. Get your build path**

    ---

    Your answers map to one of four paths: Quick Build, Standard Build, Reviewed Build, or Protected Build.

- :material-book-open-page-variant:{ .lg .middle } **3. Follow the guide**

    ---

    Step-by-step instructions while you build. The guide teaches you as you go.

    [:octicons-arrow-right-24: Build Guides](build-guides/green-quick-build.md)

- :material-check-circle:{ .lg .middle } **4. Run the checklist**

    ---

    Quick final check before going live. Confirm everything's in order.

    [:octicons-arrow-right-24: Checklists](checklists/green-checklist.md)

</div>
