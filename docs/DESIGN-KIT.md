## How to Use

When building a new tool with AI (Lovable, v0, Replit, etc.):
1. Open the Stitch project above
2. Find the screen closest to what you're building
3. Reference the design in your prompt: "Make it look like this [screenshot]"
4. Use the exact colors and fonts from the [Color Palette](/design-system/colors) page

## Figma Sync (Manual)

To create a Figma design library from the live site:
1. Install [Superposition](https://superposition.design/) (free, Adobe-backed)
2. Point it at the deployed SpringBoard URL
3. It extracts all design tokens (colors, typography, spacing) into Figma variables
4. Save as a shared Figma library for the team
