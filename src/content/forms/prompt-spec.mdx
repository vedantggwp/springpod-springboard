## Model configuration

| Parameter | Value |
|-----------|-------|
| Model | *e.g., Claude Sonnet 4, GPT-4o* |
| Temperature | *e.g., 0.3 for factual, 0.7 for creative* |
| Max tokens | *e.g., 1024* |
| Top-p | *e.g., 0.95 (leave default unless you have a reason to change)* |

<Collapsible variant="why" title="WHY this matters">

These settings directly affect output quality and consistency. A high temperature (0.7+) produces varied, creative outputs — good for brainstorming, risky for factual content. A low temperature (0.1-0.3) produces consistent, predictable outputs — better for student-facing tools and compliance-sensitive features. Document these so anyone maintaining the prompt can reproduce your results.

</Collapsible>
