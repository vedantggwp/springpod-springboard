## Common Scenarios

<Collapsible variant="tip" title="I want to build an employer engagement dashboard">

Most internal dashboards are **Green path** — the simplest process. If employers or partners will see or interact with the dashboard, it moves to **Yellow path**.

Either way, check [Data Workflows](/standards/data-workflows) before connecting to any database or pulling partner data into your tool.

</Collapsible>

<Collapsible variant="tip" title="I need to automate a reporting workflow">

Start by checking what data the workflow touches. If it includes employer PII (contact names, emails, contract details), review [Data Workflows](/standards/data-workflows) for classification rules.

Most internal reporting automation is **Green path**. If the output goes to partners, it's likely **Yellow**.

</Collapsible>

<Collapsible variant="tip" title="I'm integrating with an external partner system">

External integrations are typically **Orange path** because they involve data flowing between systems. Review [Security Standards](/standards/security) for API and access control requirements, and file a [Review Request](/forms/review-request) before starting the build.

</Collapsible>
