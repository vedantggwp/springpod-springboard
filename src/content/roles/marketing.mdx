## Common Scenarios

<Collapsible title="I want to use AI to write student-facing marketing copy" variant="tip">
  This is a **Yellow path** because the output reaches students directly. Before you start, review the [Branding Standards](/standards/branding) for voice and tone, and the [Safeguarding & Under-18s](/standards/safeguarding) standard for age-appropriate language requirements. Then follow the [Yellow Build Guide](/build-guides/yellow-standard-build).
</Collapsible>

<Collapsible title="I'm building a social media content tool" variant="tip">
  Internal drafting tools are typically **Green path**. Check the [Branding Standards](/standards/branding) to make sure generated content stays on-brand, and get the usual sign-off before publishing. Follow the [Green Build Guide](/build-guides/green-quick-build).
</Collapsible>

<Collapsible title="I need AI to generate landing page copy" variant="tip">
  If the landing page is internal-facing, this is **Green path**. If it targets under-18s, it's **Yellow path**. Either way, use prompts from the [Prompt Library](/client-config/prompt-library) for consistent brand voice, and review the [Branding Standards](/standards/branding) before going live.
</Collapsible>
