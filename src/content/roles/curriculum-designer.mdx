## Common Scenarios

<Collapsible variant="tip" title="I want to generate quiz questions from course content">

This is typically a **Green path** project — internal tooling that helps you draft questions faster. However, you must test for hallucinated facts before using any AI-generated questions with students. Check the [Prompt Management](/standards/prompts) page for testing requirements, and always have a subject-matter expert review the output before it goes live.

</Collapsible>

<Collapsible variant="tip" title="I'm building a student-facing content tool">

If students interact with your tool directly, this is likely a **Yellow path** (or higher) due to the under-18 audience. Take the [Intake Quiz](/intake/project-intake) to confirm, and read the [Safeguarding & Under-18s](/standards/safeguarding) standards before you start. Student-facing tools need additional review and safeguards.

</Collapsible>

<Collapsible variant="tip" title="I need to review AI-generated educational content">

See the [Quality Standards](/standards/quality) page for accuracy checks and review workflows. The [Prompt Management](/standards/prompts) page covers AI testing techniques — including how to catch hallucinations, check for bias, and validate factual claims before content reaches students.

</Collapsible>
