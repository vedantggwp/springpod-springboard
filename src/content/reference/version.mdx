# SpringBoard Version History

<Admonition type="info" title="Beta">
  SpringBoard is currently in beta. The first stable release will be v1.0.0.
</Admonition>

---

## v0.2.0-beta (2026-03-10)

SpringBoard becomes Springpod's own — all generic framework content replaced with company-specific guidance.

### Added

- All content rewritten for Springpod context
- Safeguarding standard for student-facing tools (under-18s)
- Data workflow requirements for tools handling personal data
- AI testing requirements in the Prompt Standard
- Role-based entry points for 5 roles: Curriculum, Ops & Partnerships, Product & Engineering, Marketing, Leadership
- Updates Feed for tracking AI developments
- Design system with Springpod brand tokens
- Dark mode with FOUC-free theme switching
- Full-text search across all pages

---

## v0.1.0 (2026-03-04)

Initial release of SpringBoard.

### Included

- Project Intake with 5-question classification
- 4 build paths: Green (Quick Build), Yellow (Standard Build), Orange (Reviewed Build), Red (Protected Build)
- Build Guides (learning mode) for all 4 paths
- Checklists (verification mode) for all 4 paths
- 4 standards: Quality, Security, Branding, Prompts
- 5 forms: Project Brief, Prompt Spec, Review Request, Incident Report, Build Log
- 4 supporting guides: First Build, Tool Selection, When to Escalate, Glossary
- Client configuration templates: Company Context, Approved Tools, Roles, Prompt Library
- Verification Log with source-linked audit trail

### Not yet included (planned for future releases)

- AI tool configuration templates
- Build registry for tracking all builds across the organisation
- Automated compliance tracking
- Web portal for interactive intake and checklist completion
